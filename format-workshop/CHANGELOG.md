# Changelog

## [0.4.0] - 2026-07-18

### Added

- 文件夹导入预检：识别可用图片并提示跳过的不支持或无效文件。
- 图片和音频工作台支持本地参数预设，可保存、应用和删除，最多保留 8 条。
- 图片和音频工作台支持恢复上次使用的设置；设置只保存经过校验的参数，不保存文件对象或预览地址。

## [0.3.0] - 2026-07-18

### Added

- 图片模式支持选择本地文件夹，文件会进入现有批量队列并显示整体进度。
- 文件夹批次复用部分成功结果、失败文件重试和 ZIP 打包下载流程。

### Changed

- 文件夹入口仅在图片模式显示，音频、视频和 PDF 保持原有单任务流程。

## [0.2.0] - 2026-07-18

### Added

- 音频裁剪、音量调整、响度标准化、播放速度和淡入淡出。
- 图片旋转、水平/垂直翻转、文字水印和安全的批量重命名。
- 文件尺寸、时长、MIME 类型和修改时间信息。
- PWA 安装入口、应用外壳与本地转换资源的离线缓存。

### Changed

- 本地工具数量从 21 项扩展到 32 项。
- 图片和音频参数面板改为更完整的响应式编辑工作台。
