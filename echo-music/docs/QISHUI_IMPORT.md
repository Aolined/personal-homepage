# 汽水音乐分享导入

## 支持范围

Echo Music 首页的“汽水导入”接受以下官方分享地址：

- `https://qishui.douyin.com/s/...`
- `https://music.douyin.com/qishui/share/...`

支持公开分享页中的单曲和歌单。UGC 视频、MV、需要登录才能读取的页面以及非官方域名不会作为歌曲导入。

## 工作方式

1. Echo Music 逐级校验汽水官方短链跳转。
2. 仅解析公开页面中的 SSR JSON 和 meta 数据，不执行页面脚本。
3. 使用歌曲名和歌手名查询现有网易云音乐、QQ 音乐接口。
4. 按标题、歌手和版本特征评分，过滤低可信度、Remix、翻唱或现场版误匹配。
5. 用户确认后加入队列或立即播放匹配结果。

单次最多处理前 24 首歌曲，并以 2 路并发执行匹配。未匹配项目会在界面中明确显示。

## 不包含的能力

- 汽水音乐账号登录
- 收藏或歌单双向同步
- 汽水会员权益读取
- 汽水原始音频代理或下载
